## Screenshot Guidelines

### Technical Specifications
- **Resolution**: 1920x1080 (Full HD)
- **Format**: PNG with transparency support
- **Compression**: Optimized for web (lossless compression)
- **Color Profile**: sRGB for web compatibility

### Content Guidelines
- **No Personal Data**: Ensure no sensitive user information is visible
- **Clean Interface**: Remove any debugging information or console logs
- **Consistent Branding**: Use consistent colors and typography
- **Professional Appearance**: High-quality, professional presentation

### File Naming Convention
```
{feature}_{state}_{device}.png

Examples:
- login_desktop.png
- dashboard_mobile.png
- recommendations_tablet.png
```
